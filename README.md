# Networking Fundamentals

## Overview

This repository contains structured notes on core computer networking concepts, developed as part of my foundational learning in preparation for advanced studies in cybersecurity and penetration testing.

The goal is to build a clear understanding of how data moves across networks, how systems communicate, and how these mechanisms form the basis for both modern internet infrastructure and cybersecurity analysis.

---

## Purpose

The purpose of this repository is to:

- Strengthen foundational networking knowledge
- Develop clear and structured technical communication
- Build a reference base for future cybersecurity and penetration testing studies
- Demonstrate understanding of core networking concepts through simplified explanations

---

## Topics Covered

This repository currently includes the following core areas:

- TCP/IP model and data flow
- DNS resolution process
- Subnetting and IP addressing
- HTTP and HTTPS communication

Each topic is explained in a simplified, structured format to support both understanding and revision.

Each topic is written with the following approach:

This ensures that concepts are not only memorized, but understood in context.
