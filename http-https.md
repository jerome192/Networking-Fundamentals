# HTTP vs HTTPS Explained

## 1. HTTP
HTTP is used to transfer data between browser and server.

Problem:
- data is not encrypted

---

## 2. HTTPS
HTTPS adds encryption using TLS.

---

## 3. Difference

HTTP → readable data  
HTTPS → encrypted data

---

## 4. How it works

1. Client connects
2. TLS handshake occurs
3. Secure communication begins

---

## 5. Why it matters in cybersecurity

Without HTTPS:
- data can be intercepted
- passwords can be stolen
