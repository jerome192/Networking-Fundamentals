# Subnetting Explained

## 1. What is subnetting?
Subnetting divides a large network into smaller networks.

---

## 2. Why it is used
- better performance
- better security
- efficient IP usage

---

## 3. Example

IP: 192.168.1.0/24

This means:
- 256 total addresses
- 254 usable hosts

---

## 4. Key idea

Network part + Host part

---

## 5. Why it matters in cybersecurity

Subnetting helps:
- isolate systems
- control access
- limit attack spread
